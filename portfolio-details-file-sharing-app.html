AWS S3 based File Sharing App

Project information

  • Category: Full Stack Development
  • Project date: 30 January, 2021
  • Project URL: https://sukriti-file-sharing.netlify.app/
  • Github Frontend URL: https://github.com/mishrasukriti/filesharing-frontend
  • Github Backend URL: https://github.com/mishrasukriti/fileSharingApp-backend

Designed a File sharing application to share the files using Nodejs as backend and Reactjs as frontend. A unique URL is generated for each file uploaded and the files which are to be shared are stored in AWS S3 (Simple Storage Service). Also, application asks for email to send the unique URL. Generated mail contains the file uploaded unique URL.
